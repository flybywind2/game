# 몽글 놀이터 19,000원 출시 전 아이 관찰 테스트

## 대상과 준비

- 36~48개월 아이 최소 3명에게 각각 따로 진행합니다.
- 평소 사용하는 휴대전화나 태블릿에서 전체 화면으로 엽니다.
- 처음 10분 동안 보호자는 화면을 누르거나 정답을 알려주지 않습니다.
- 아이에게는 “몽글이와 놀아볼까?”라고 한 번만 말합니다.
- 보호자 공간의 `10분 혼자 놀이 관찰`에서 `관찰 준비하기`를 누릅니다.
- 준비 화면이 뜨면 휴대전화를 아이에게 건네고 아이가 `놀이 시작!`을 직접 누르게 합니다.

## 10분 관찰 순서

1. 준비 화면에서 아이가 `놀이 시작!`을 스스로 누르는지 봅니다. 첫 시작 시간은 자동으로 기록됩니다.
2. 관찰 모드가 `알록달록 과일` → `쌍둥이 그림 찾기` → `뜨거워요, 멈춰요`를 차례로 이어 줍니다.
3. 아이가 7초 이상 멈춰 자동 도움이 나온 경우에도 보호자는 기다립니다.
4. 잘못 눌러도 정답을 알려주지 않고 다시 시도하는지 봅니다.
5. 놀이 두 개를 끝낸 뒤 아이가 스스로 다른 놀이를 고르는지 봅니다.
6. 종료 후 `보호자와 결과 보기`를 눌러 대신 누른 횟수·스크롤 혼동·내용 불일치·추가 선택·다시 하기 반응을 기록합니다.
7. 아이 3명의 관찰을 마치면 `결과 복사`를 눌러 자동 지표와 출시 기준 판정을 함께 보관합니다.

## 19,000원 출시 통과 기준

아래 조건을 모두 만족해야 판매 준비 완료로 판단합니다.

- 3명 중 2명 이상이 30초 안에 첫 놀이를 시작합니다.
- 3명 모두 보호자의 화면 조작 없이 최소 2개 놀이를 끝냅니다.
- 아이 한 명당 자동 도움은 10분 동안 3회 이하입니다.
- 드래그와 페이지 스크롤을 혼동한 사례가 전체 테스트에서 1회 이하입니다.
- 3명 중 2명 이상이 스스로 추가 놀이를 선택합니다.
- 3명 중 2명 이상이 다시 하고 싶다고 표현합니다.
- 정답과 게임 내용이 어긋난 사례가 한 건도 없습니다.

조건 하나라도 통과하지 못하면 해당 장면의 게임 이름, 아이가 멈춘 행동, 자동 도움 횟수를 함께 기록한 뒤 수정하고 다시 테스트합니다.
